# Ecovacs GOAT G1

Mower-only Home Assistant custom integration for ECOVACS GOAT lawn mowers.

This integration is intentionally narrow and was built for the GOAT G1-800. It avoids `deebot-client` and vacuum/XMPP code paths, using app-style N-GIoT commands and MQTT push updates instead.

See the repository root `README.md` for installation, support scope, safety notes, and acknowledgements.
